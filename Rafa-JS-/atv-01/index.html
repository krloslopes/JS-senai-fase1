Atalhos gerais

vamos ver alguns atalhos legais para VS code que irao facilitar o nosso uso diario do mesmo.

alt+z:quebra automatica de linha

alt+setas para cima ou para baixo: movem a seleção para cima ou para baixo no arquivo sem estar selecionado

shift+alt+setas cima/baixo: copia a seleção para cima/para baixo no arquivo,se nada estiver selecionado copia a linha que o cursor esta.

ctrl+; : comentario mais descomentario
